## 2. Улучшения UX

### Улучшение 1 — Нижняя навигация с FAB по центру

**ДО:** Навигация через иконки в AppBar — неочевидно, не соответствует Material Design.

**ПОСЛЕ:** `BottomAppBar` с 5 элементами: Главная · Статистика · **➕ FAB** · Кошельки · Категории. При переключении с вкладки кошельков `_refreshKey++` пересоздаёт `HomeScreen` и `StatisticsScreen` с актуальным активным кошельком.

📸 **Скриншот ДО:** главный экран — AppBar с иконками навигации, нижней панели нет.
📸 **Скриншот ПОСЛЕ:** главный экран — внизу панель с 5 иконками, по центру круглая teal-кнопка «+».

### Улучшение 2 — Swipe to Delete

**ДО:** Удаление только через `onLongPress` — скрытый жест.

**ПОСЛЕ:** `Dismissible` — свайп влево показывает красный фон с иконкой корзины и диалог подтверждения. `onLongPress` для редактирования сохранён.

📸 **Скриншот ДО:** список операций — обычные тайлы без подсказок.
📸 **Скриншот ПОСЛЕ:** момент свайпа — красный фон с иконкой удаления.

### Улучшение 3 — Empty State CTA

**ДО:** При отсутствии операций — только серый значок и текст, нет подсказки что делать.

**ПОСЛЕ:** Добавлена кнопка `FilledButton` «Добавить операцию» прямо в блоке пустого состояния. FAB с главного экрана убран.

📸 **Скриншот ДО:** пустой кошелёк — только иконка и текст без кнопки.
📸 **Скриншот ПОСЛЕ:** добавлена teal-кнопка «Добавить операцию» под текстом.
